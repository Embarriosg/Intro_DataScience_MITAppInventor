# Introduction to Data Science with MIT App Inventor

This repository contains introductory projects and exercises focused on core Data Science concepts, developed using MIT App Inventor as part of the Tecnolochicas program.  
It serves as an introductory portfolio showcasing the projects completed over a two-week period, emphasizing hands-on learning, data-driven thinking, and the application of computational tools in an educational context.

## Tools & Technologies
- MIT App Inventor
- Basic Data Science concepts
- Simple machine learning models
- Teachable Machine (Google) for image classification
- Spreadsheet tools (Google Sheets) for basic data exploration
- CloudDB within MIT App Inventor for basic data persistence
- AI model APIs (Ollama, Gemini)
